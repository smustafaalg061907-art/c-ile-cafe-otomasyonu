# c-ile-cafe-otomasyonu
C#, DevExpress ve Entity Framework ile N-Tier (Katmanlı) Mimari yapısında geliştirilmiş profesyonel Cafe Otomasyonu. SOLID prensiplerine uygun; dinamik masa yönetimi, hızlı sipariş, stok takibi ve gelişmiş raporlama modüllerini içeren, modern UI/UX deneyimine sahip yüksek performanslı bir kurumsal masaüstü çözümüdür.
